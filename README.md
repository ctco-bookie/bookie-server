# bookie-server

## CI

Travis CI: [![Build Status](https://api.travis-ci.org/ctco-dev/bookie-server.svg?branch=master)](https://travis-ci.org/ctco-dev/bookie-server)

## Required Software
- npm
- node >= 6.x

## Installation
`$ npm i`

## Development Mode
`$ npm start`

## Tests

### Run tests once
`$ npm t`

### Run tests in watch mode
`$ npm run tw`
