# bookie-server

## Required software
- yarn
- node >= 4.x

## Installation
`$ yarn`

## Development mode
`$ npm start`

## Tech stack

- Koa 2
  - [API](https://github.com/koajs/koa/blob/v2.x/docs/api/index.md)
  - [Guide](https://github.com/koajs/koa/blob/v2.x/docs/guide.md)
  - [TC39 Async/Await proposal](https://tc39.github.io/ecmascript-asyncawait/)
